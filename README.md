# User Management System
A Java Web Application developed using JSP, Servlets, JDBC, and MYSQL following the MVC Architecture and DAO Design Pattern. 

## Technologies Used
- JAVA
- JSP
- Servlets
- JDBC
- MYSQL
- APache Tomcat 10.1
- MVC Architecture
- DAO Pattern
- Eclipse IDE
- Git & GitHub

## Features
- User Registration
- User Login
- User Logout
- Display Users
- Search User
- Update User
- Delete User
- Session Management
